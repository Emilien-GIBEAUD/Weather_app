# Weather_app 
### ![FR](https://flagcdn.com/w20/fr.png) Français
Une application météo très simple.

Cette application donne la météo d'une ville spécifique en utilisant l'API open-meteo ([https://open-meteo.com/en/docs](https://open-meteo.com/en/docs)). La météo est mise à jour toutes les heures.

<br>

L'application est accessible sur la GitHub Page suivante :

# Weather_app
### ![EN](https://flagcdn.com/w20/gb.png) English

A very simple web weather application.

This application give the weather for a specific town using the API open-meteo ([https://open-meteo.com/en/docs](https://open-meteo.com/en/docs)). The weather is updated each hour.

You can access to the app with this GitHub Page :
